# 读取VASP分子动力学的脚本教程
## 1、确定模块正常安装，并打印帮助文档。
```
python read_vasp_MD.py -h
```

输出：
```
Read outputs of a VASP run.

optional arguments:
  -h, --help            show this help message and exit
  -t TAG [TAG ...], --tag TAG [TAG ...]
                        Tags for printing out and save. E: Energy (AIMD); T: Temperature; D: Distance
  -p PAIR [PAIR ...], --pair PAIR [PAIR ...]
                        Distances between atomics pairs. Atomic index starts from ONE.
  -s SAVE, --save SAVE  Save the output to the disk.
  -d DELETE, --delete DELETE
                        Delete/hide the data of certain frame. Frame index from ONE.
  -NoHead NOHEAD, --NoHead NOHEAD
                        Hide the head line of the output
```
各选项解释：
```-h```: 打印帮助文档。
```-t``` 或 ```--tag```：选择要读取的信息。包括```['Step', 'T', 'E', 'F', 'E0', 'EK', 'SP', 'SK', 'Dist', 'Force']```。其中```force```暂时没有写，```Step```为默认输出，且可不规定```Step```。默认值为：```['T', 'E', 'Dist']```
```-p``` 或 ```--pair```：如果```-t```选项包含了```Dist```，需要同时用```-p```选项规定原子对，原子序号从```1```开始。可同时规定多对，如 ```1-2``` ```2-3``` ```1-3```。该选项无默认值。
```-s``` 或 ```--save```: 是否要将读取的信息保存到文件中。默认为 ```F```。如果是```T```，信息将不打印到屏幕，而是保存到文件。
```-d``` 或 ```--delete```: 删除指定MD步的数据。原子序号从```1```开始。默认为```None```（不删除）。
-NoHead 或 --NoHead: 隐藏文件头。默认为False（不隐藏）
